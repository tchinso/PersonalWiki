# 글자 크기 및 글자 색
사실 쭉 생각하고 있고, 개선한다면 이것부터 개선할듯? 문제는 어떻게 문법을 구성할 것인가에 대해서 고민중임.
다른 텍스트 서식과 비교해서 어느 정도 합의된 문법도 없다시피하고...
사실 애초에 글자 크기는  지금 상태에서도 디스코드에서 쓰이는 마냥 # 헤더를 이용해 대체할 수 있긴 함.
(헤더의 본래 목적이 글자 크기 조정이 아닐 지라도, 디스코드 같은 데서는 헤더 문법을 그런 식으로 사용함.)
PersonalWiki도 디스코드 식 유행을 감안하여 제작된거라 (Spoiler 문법도 그렇고) 꼭 추가해야하나? 는 회의감은 있음
