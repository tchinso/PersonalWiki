# 글자 크기 및 글자 색
CommonMark, GFM, Obsidian, MKDocs, Discord 에서 모두 글자크기/글자색 같은 표현 문법을 지원하지 않음.
Pandoc, Rentry 등에서는 지원하긴 하는데 메이저 문법에서 지원하지 않는 걸 굳이 추가해야하나 싶음

# 고급 표기능
표 문법은 어떻게 수정하더라도 끔찍해지기 때문에 그냥 markdown 기본 표 테이블 문법 그대로 놨둘 것임
